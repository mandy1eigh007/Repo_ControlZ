# Repo ControlZ — Project Brief

## What this is
Repo ControlZ is a **living manual** for AI Build Foreman workflows.

It stores repeatable instructions, prompts, templates, and troubleshooting workflows for managing AI coding agents across tools like Codex, VS Code Chat / Copilot, Replit agents, and GitHub/Codespaces.

## Primary user
- Mandy

## Purpose
- Keep durable, repo-based instructions for controlling AI agents
- Provide reusable prompts and templates
- Capture lessons learned and troubleshooting playbooks

## Important scope boundary
This repository is **not an app** and should not become an app unless the user explicitly changes that scope later.
